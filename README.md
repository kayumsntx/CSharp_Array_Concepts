 Features:

- DeclareArray() → Array declaration and initialization
- ArrayInstance() → Explicit and implicit array creation
- AnonymousTypesArray() → Array of anonymous types
- AccessingArray() → Accessing array elements
- PassingThroughArray() → Passing arrays to methods
- CopyingArray() → Different array copying techniques
- SingleDimensionalArray() → One-dimensional array
- MultiDimensionalArray() → Multi-dimensional array
- JaggedArray() → Jagged array (array of arrays)
- Array.Sort()
- Array.Reverse()
- LINQ (Where, Select)
